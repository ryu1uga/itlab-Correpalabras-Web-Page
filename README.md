# Explicación
Este es una página web de presentación de la aplicación móvil **Correpalabras**. El proyecto consiste en una aplicación digital que busca promover la lectura infantil en el hogar aprovechando los teléfonos celulares de los padres de familia. Se dirige principalmente a niñas y niños de segundo, tercero y cuarto grado de primaria.

# Estructura de archivos
- 📁 src
  - 📁 components
    - 📁 Footer --> El footer que va al final de todas las páginas
    - 📁 ImageSlider --> El Slider de imágenes de la página principal
    - 📁 Navbar --> Es la barra de navegación
    - 📁 TestimonialSlider --> El slider de testimonios de la página principal
  - 📁 pages
    - 📁 AcercaDe --> Es la página Acerca De
    - 📁 CreaCuentos --> Es la página de Crea Cuentos
    - 📁 FichasComprension --> Es la página de Comprensión Lectora
    - 📁 Home --> Es la página principal
    - 📁 Investigaciones --> Es la página de Investigaciones
  - 📄 App.jsx --> Crea el router y exporta App
  - 📄 index.css --> Es donde se define los tamaños de fuente responsivos y los colores de texto para las páginas
  - 📄 index.jsx --> Renderiza App

# Cómo empezar
1. Clona el repositorio:
  `git clone <url-del-repositorio>`

2. Instala las dependencias:
  `npm install`

3. Inicia el servidor de desarrollo:
  `npm run dev`

¡Ya está listo para usar!
